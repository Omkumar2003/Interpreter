Learn the Language

Master the OM programming language with Hindi-inspired keywords

Variables

Declare variables using definekar:

definekar x = 5;
definekar y = 10;

Functions

Create functions using fn:

definekar add = fn(a, b) {
    a + b
};
add(2, 3);

Conditionals

Use agar (if) and warna (else):

agar (x > 5) {
    10
} warna {
    20
}

Recursion

Create recursive functions:

definekar factorial = fn(n) {
    agar (n == 0) {
        1
    } warna {
        n * factorial(n - 1)
    }
};
factorial(5);
